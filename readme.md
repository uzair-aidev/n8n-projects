## Projects

*   **Email Classifier:** This project uses AI to classify incoming emails and adds a label to them in Gmail. You can find the workflow in the `Email_classifier` directory.

## How to Use

To use these projects, you will need to have a running n8n instance. You can then import the `.json` file for the project you want to use.

1.  Go to your n8n instance.
2.  Click on the "New" button and then "Import from File".
3.  Select the `.json` file for the project you want to import.
4.  Configure the credentials and other settings as needed.
